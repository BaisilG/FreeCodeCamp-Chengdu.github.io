> [【提交入口】](https://github.com/FreeCodeCamp-Chengdu/FreeCodeCamp-Chengdu.github.io/new/hexo/source/_posts/Activity/)

Front-matter 规范：

```yaml
title:  # 活动主题（翻译成英文后设为本文件的文件名）
date:  # 发起日期
categories:
  - Activity
  # 二级分类：Salon、Workshop 或 Conference
tags:
  # online 或 offline
  # 与内容相关的更多标签
toc: true

# Activity meta
start:  # 活动开始时间
end:    # 活动结束时间
description:  # 活动简介
mentors:
  # 讲师、教练们的 GitHub 账号
workers:
  # 组织者、志愿者们的 GitHub 账号
```
